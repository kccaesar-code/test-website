🎯 為什麼需要學習提示詞？ 🎯 Why Learn Prompt Engineering?

在AI時代，掌握有效的提示詞技巧就像掌握了一門新的教學語言。好的提示詞可以讓AI更好地理解你的需求，提供更準確、更有價值的教學資源。 In the AI era, mastering effective prompt engineering is like learning a new teaching language. Good prompts help AI better understand your needs and provide more accurate, valuable teaching resources.

本頁面提供實用的提示詞範本和技巧，幫助教師們快速上手AI教學工具。 This page provides practical prompt templates and techniques to help teachers quickly master AI teaching tools.

💡 提示詞基本原則 💡 Basic Prompt Principles

1. 明確具體：清楚說明你想要什麼 1. Be Clear & Specific: Clearly state what you want

2. 提供上下文：告訴AI你的教學情境 2. Provide Context: Tell AI about your teaching situation

3. 設定角色：讓AI扮演特定角色（如資深教師） 3. Set Roles: Let AI play specific roles (e.g., experienced teacher)

4. 分步驟要求：複雜任務分解為小步驟 4. Break Down Tasks: Divide complex tasks into small steps

5. 提供範例：展示你期望的格式或風格 5. Provide Examples: Show the format or style you expect

🎨 進階技巧 🎨 Advanced Techniques

1. 溫度控制：調整AI的創造性程度 1. Temperature Control: Adjust AI's creativity level

2. 種子設定：確保結果的一致性 2. Seed Setting: Ensure consistency of results

3. 系統提示：設定AI的整體行為模式 3. System Prompts: Set AI's overall behavior pattern

4. 少樣本學習：提供少量例子讓AI學習模式 4. Few-shot Learning: Provide few examples for AI to learn patterns

5. 鏈式思考：要求AI展示推理過程 5. Chain-of-Thought: Ask AI to show reasoning process

📝 最佳實踐示範 📝 Best Practice Example

# 角色：資深中學數學教師 # 任務：設計一份關於二次函數的教案 # 年級：中三學生 # 時間：45分鐘課堂 # 要求： 1. 包含學習目標 2. 設計互動活動 3. 提供練習題（分基礎、進階） 4. 建議評估方法 5. 使用生活化例子 # Role: Experienced Secondary Math Teacher # Task: Design a lesson plan about quadratic functions # Grade: Form 3 students # Time: 45-minute class # Requirements: 1. Include learning objectives 2. Design interactive activities 3. Provide practice questions (basic & advanced) 4. Suggest assessment methods 5. Use real-life examples

這個提示詞明確、具體，提供了完整的上下文，讓AI能生成高質量的教學資源。 This prompt is clear, specific, and provides complete context, enabling AI to generate high-quality teaching resources.

📋 試卷生成技巧 📋 Exam Paper Generation Tips

1. 明確考試目標：診斷性、形成性還是總結性評估？ 1. Clarify Exam Purpose: Diagnostic, formative, or summative assessment?

2. 設定評分標準：提前確定評分維度和權重 2. Set Scoring Criteria: Determine scoring dimensions and weights in advance

3. 平衡題型：客觀題和主觀題的適當比例 3. Balance Question Types: Appropriate ratio of objective and subjective questions

4. 難度分層：基礎題、中等題、挑戰題的比例 4. Difficulty Stratification: Ratio of basic, intermediate, and challenging questions

5. 時間管理：確保題目數量與考試時間匹配 5. Time Management: Ensure question quantity matches exam time

6. 覆蓋全面：確保試卷覆蓋所有教學重點 6. Comprehensive Coverage: Ensure exam covers all teaching key points

📖 教案設計類 📖 Lesson Plan Design

1. 單元教案設計 1. Unit Lesson Plan Design

設計完整的教學單元，包含多節課的連貫安排 Design complete teaching units with coherent arrangement of multiple lessons

請以[科目]教師的身份，為[年級]學生設計一個關於[主題]的教學單元。 要求： 1. 單元總時長：[X]節課 2. 每節課的教學目標 3. 教學活動設計（包含互動環節） 4. 教學資源建議 5. 評估方式 6. 跨學科連結（如適用） As a [Subject] teacher, design a teaching unit about [Topic] for [Grade] students. Requirements: 1. Unit total duration: [X] lessons 2. Teaching objectives for each lesson 3. Teaching activity design (including interactive sessions) 4. Teaching resource suggestions 5. Assessment methods 6. Cross-disciplinary connections (if applicable)

2. 差異化教學設計

為不同能力的學生設計差異化教學活動

請設計一個差異化教學方案，主題是[教學主題]。 要求針對三種不同能力的學生： 1. 基礎組：需要更多支持和引導 2. 標準組：能完成基本要求 3. 進階組：需要挑戰性任務 請為每組設計： - 具體學習目標 - 教學活動 - 評估標準 - 支援策略

✍️ 試題設計類

1. 選擇題設計

設計高質量的選擇題，包含干擾項分析

請為[科目][年級]的[主題]設計10道選擇題。 要求： 1. 涵蓋不同認知層次（記憶、理解、應用、分析） 2. 每題4個選項，包含1個正確答案和3個有價值的干擾項 3. 說明每個干擾項的設計意圖 4. 提供參考答案和解析 5. 標註題目難度（易/中/難）

2. 開放式問題設計

設計促進批判性思考的開放式問題

請設計5個關於[主題]的開放式問題，用於促進學生的批判性思考。 要求： 1. 問題應引導學生進行分析、評估和創造 2. 每個問題附帶： - 預期的思考過程 - 可能的回答方向 - 評分準則（如適用） 3. 問題應分層次，從基礎到進階

3. 完整試卷生成

生成包含多種題型的完整考試試卷

請為[科目][年級]的[單元名稱]生成一份完整的考試試卷。 試卷要求： 1. 考試時間：[X]分鐘 2. 總分：[X]分 3. 題型分佈： - 選擇題：[X]題，每題[X]分，共[X]分 - 填空題：[X]題，每題[X]分，共[X]分 - 簡答題：[X]題，每題[X]分，共[X]分 - 論述題/應用題：[X]題，每題[X]分，共[X]分 具體要求： 1. 試卷結構： - 封面：科目、年級、考試時間、總分 - 考生須知 - 各部分題目 - 答題紙格式建議 2. 題目設計： - 涵蓋單元所有重點內容 - 難度分佈：易30%、中50%、難20% - 認知層次分佈：記憶20%、理解30%、應用30%、分析評價20% 3. 評分標準： - 每題提供詳細評分準則 - 選擇題：正確答案和解析 - 填空題：可接受答案範圍 - 簡答題：要點和分值分配 - 論述題：評分維度（內容、結構、表達等） 4. 附加材料： - 教師版：包含所有答案和解析 - 學生版：僅題目 - 建議考試時間分配 請以專業的試卷格式呈現，確保題目清晰、無歧義。

4. 單元測驗生成

生成針對特定教學單元的小測驗

請為[科目][年級]的[單元名稱]生成一份單元測驗。 測驗要求： 1. 測驗時間：15-20分鐘 2. 總分：20分 3. 題型：選擇題、填空題、簡答題組合 4. 題目數量：8-10題 具體要求： 1. 內容覆蓋： - 涵蓋單元核心概念 - 重點難點均衡分佈 - 包含基礎知識和應用題 2. 題目設計： - 選擇題：4-5題，測試基本概念理解 - 填空題：2-3題，測試關鍵術語記憶 - 簡答題：1-2題，測試應用能力 3. 評分標準： - 每題明確分值 - 提供參考答案 - 簡答題提供評分要點 4. 教學用途： - 可用於課堂小測驗 - 可用於課後練習 - 可用於學習效果評估 請確保題目難度適中，適合單元學習後的評估。

5. HKDSE/公開考試模擬試卷

生成符合公開考試標準的模擬試卷

請生成一份符合HKDSE [科目]考試標準的模擬試卷。 試卷要求： 1. 嚴格按照HKDSE考試格式 2. 考試時間：[與真實考試一致] 3. 題型分佈：與歷屆試題一致 4. 難度水平：與真實考試相當 具體要求： 1. 試卷結構： - 卷別（如卷一、卷二） - 各部分題目數量和分值 - 考試指示語（與考評局格式一致） 2. 題目設計： - 參考最近5年歷屆試題趨勢 - 涵蓋課程所有重點單元 - 包含熱門題型和新增題型 - 難度分佈符合考試標準 3. 評分標準： - 提供詳細評分方案（Marking Scheme） - 包含等級描述（Level Descriptors） - 提供樣本答案（Sample Answers） - 標註常見錯誤和注意事項 4. 附加材料： - 考生表現分析 - 教學建議 - 備考策略 - 時間管理建議 請確保試卷的專業性和準確性，適合用作模擬考試。

👥 課堂活動類

1. 分組活動設計

設計合作學習的分組活動方案

請設計一個關於[主題]的分組合作學習活動。 要求： 1. 活動名稱和目標 2. 分組方式（每組人數、角色分配） 3. 活動流程（時間分配、具體步驟） 4. 所需材料 5. 教師指導要點 6. 評估方法（個人和組別評估） 7. 常見問題及解決方案

2. 遊戲化學習設計

將學習內容轉化為遊戲形式

請將[學習內容]設計成一個課堂遊戲。 要求： 1. 遊戲名稱和簡介 2. 遊戲目標和學習目標的對應 3. 遊戲規則（清晰易懂） 4. 所需材料和準備工作 5. 遊戲流程（含時間安排） 6. 計分和獎勵機制 7. 遊戲後的討論問題

📊 評估回饋類

1. 評分準則設計

設計清晰明確的評分準則（Rubric）

請為[作業類型]設計一個評分準則（Rubric）。 要求： 1. 評估維度（如內容、組織、表達、創意等） 2. 每個維度的評分等級（如優秀、良好、合格、待改進） 3. 每個等級的具體描述 4. 總分計算方式 5. 給學生的使用說明 6. 教師使用注意事項

2. 建設性回饋生成

為學生作業提供具體、建設性的回饋

請為以下學生作業提供建設性回饋： [在此貼上作業內容或描述] 要求回饋包含： 1. 肯定優點（具體指出做得好的地方） 2. 改進建議（具體、可操作） 3. 學習資源推薦（相關的補充材料） 4. 下一步行動建議 5. 鼓勵性結語 語氣：積極、支持、具體

🚀 進階應用建議

1. 建立個人提示詞庫：收集和整理對你最有用的提示詞。

2. 迭代優化：根據AI的回應調整和改進你的提示詞。

3. 分享與合作：與同事分享有效的提示詞，共同提升教學效率。

4. 記錄使用效果：記錄哪些提示詞最有效，建立最佳實踐。

5. 結合其他工具：將AI生成的內容與其他教學工具結合使用。

6. 試卷生成專業化：

  • 建立題庫系統：將生成的試題分類存儲，建立可重用的題庫
  • 難度分層：根據學生水平調整試題難度分佈
  • 題型多樣化：結合選擇題、填空題、簡答題、論述題等多種題型
  • 評分標準化：制定清晰的評分準則，確保評分一致性
  • 試卷分析：使用AI分析學生表現，改進試題設計
  • 版本管理：為不同班級或學年創建不同版本的試卷

7. 試卷生成最佳實踐：

  • 先生成大綱，再填充具體題目
  • 確保題目覆蓋所有教學重點
  • 檢查題目之間的獨立性
  • 驗證答案的正確性和完整性
  • 考慮考試時間和題目數量的平衡
  • 提供不同難度層次的題目
